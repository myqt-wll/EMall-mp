## 案例

基本使用方法
```html
<yz-swipe imagelist="{{imagelist}}" autoplayitem="3000"></yz-swipe>
```

## 属性

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| imagelist | 轮播图地址数据 | 	Array | null |
| autoplayitem | 轮播时长 | 	Number | 2000 |
